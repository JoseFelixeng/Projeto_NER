# Projeto: Analise dos textos CT UFRN 
Projeto desenvolvido para a disciplina de algoritmo e estrutura de dados II, com o objetivo de colocar em prática os contéudos vistos em aula.
### Integrantes
- JOSÉ FELIX RODRIGUES ANSELMO
- LUCAS HENRIQUE ALVES DE QUEIROZ
- CARLOS EDUARDO NASCIMENTO MORAIS 

### Objetivo

Analisar os trabalhos de conclusão de curso(TCC) do período de Janeiro de 2025 do CT UFRN, extraindo os texto de .PDF para .txt. Criar mecânismos de limpeza dos textos, aplicar ferramentas de PLN e logo após criar grafos de co-ocorrencias gerados apartir dos dados desses trabalhos, e por fim gerar imagens que demonstrem essas relações, para facilitar a visualização e analise.


### Metodologia
- Coleta de dados do periodo de Janeiro de 2025
- Converter os textos em .pdf para .txt usando PyMuPDF
- Retirar as instâncias invalidas após a conversão usando Regex.
- Usar NER manual de 
- Construção do grafo
- Criar sentenças, paragrafos e k-caracteres
- Atraves desse analisar as densidades, distancias, Nos e arestas de um grafo
- Utilizar a Biblioteca pyvis para criar uma pagina html interativa para demonstrar as coneções entre os elementos
- Criar figuras que monstram os dados das analises feitas
