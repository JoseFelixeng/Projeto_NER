## Objetivo
Analise do minimundo dos TCCs desenvolvidos durante o periodo de janeiro de 2025, pelos alunos do CT UFRN, e aplicar tecnicas de limpeza de ruídos no texto usando o Regex e também aplizando metodos de NER usando grafos para a representação dos trabalhados.
 
## Metodologia
- Coleta de dados do periodo de Janeiro de 2025
- Converter os textos em .pdf para .txt usando PyMuPDF
- Retirar as instâncias invalidas após a conversão usando Regex.
- Construção do grafo
- Criar sentenças, paragrafos e k-caracteres
- Atraves desse analisar as densidades, distancias, Nos e arestas de um grafo
- Utilizar a Biblioteca pyvis para criar uma pagina html interativa para demonstrar as coneções entre os elementos
- Criar figuras que monstram os dados das analises feitas
